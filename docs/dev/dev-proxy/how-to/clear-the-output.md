# Clear the output

> **At a glance**  
> **Goal:** Clear Dev Proxy console output  
> **Time:** 1 minute  
> **Plugins:** None  
> **Prerequisites:** [Set up Dev Proxy](../get-started/set-up.md)

To clear the proxy output in your command prompt, press <kbd>c</kbd> on your keyboard.
